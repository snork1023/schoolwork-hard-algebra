# Welcome to Kepler

Our platform provides a smooth, reliable experience where users can access a wide range of unblocked games and a fast, built‑in search experience.

## Project info

**URL**: https://schoolwork-hard-algebra.vercel.app/

## Running through GitHub Codespaces (FIX IN PROGRESS)

- Click the green Code button at the top right of the repo
- Choose the Codespaces tab
- Select Create codespace on main
- Open the terminal
- Run 'npm install' to install the proper dependencies
- Then run 'npm run dev' and open the port
