# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| > 2.0   | :white_check_mark: |
| < 2.0   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability in this project, we appreciate your help in disclosing it responsibly.

### How to Report
Please report vulnerabilities through one of the following methods:

- **GitHub Private Vulnerability Reporting**

### What to Expect
- We will acknowledge your report within **48 hours**.
- We will provide updates at least **once per week** while the issue is being investigated.
- If the vulnerability is confirmed, we will work to release a fix as quickly as possible.
- We will notify you when the issue is resolved, and we may credit you in the release notes if you wish.

### Responsible Disclosure
Please do not publicly disclose the vulnerability until we have released a fix. This helps protect users of the project.

Thank you for helping keep this project secure.
